# Security Policy

## Supported versions

The latest release receives security fixes. Older versions are not maintained.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Report vulnerabilities privately via GitHub's
[security advisories](https://github.com/Razee4315/Pin-It/security/advisories/new),
or by email to <saqlainrazee@gmail.com>. Include steps to reproduce and the
affected version. You can expect an acknowledgement within a few days.

PinIt is a local Windows utility with no network functionality; it reads/writes
only its own config and log under `%LOCALAPPDATA%\PinIt`.
